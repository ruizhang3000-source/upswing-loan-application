Yes. Best option: Azure Static Web Apps under your Microsoft/Office 365 account.

Why: your form is just an HTML file, and Azure Static Web Apps gives you a secure website with custom domain and free SSL certificate. Microsoft supports custom domains on Static Web Apps. Microsoft Learn

Use something like:

apply.upswingcap.com.au

Basic steps:

  1. Create an Azure account using your Microsoft 365 admin login.

  2. Go to Azure Portal → Static Web Apps → Create.

  3. Upload the HTML form to GitHub, or use Azure deployment.

  4. Add custom domain:
    apply.yourdomain.com

  5. In your domain DNS, add the CNAME record Azure gives you.

  6. Azure validates the domain and creates HTTPS/SSL automatically for the custom domain. Microsoft Learn

  7. Test the form on mobile and desktop.

Do not use plain Azure Blob Storage unless you also set up CDN/Front Door, because Azure Storage static sites do not natively support HTTPS on custom domains. Microsoft Learn

My recommended setup for Upswing Capital:

www.upswingcap.com.au = main website
apply.upswingcap.com.au = bilingual loan application form
clients.upswingcap.com.au = future client portal
